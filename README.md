##How to build
1)  Clone the repository
```
$ git clone https://github.com/cuiwei121/Joke.git
```
2) Open the workspace in Xcode
```
$ open "Joke.xcworkspace"
```
3) Compile and run the app in your simulator or iOS device

##Requirements
* Xcode 7.2+
* iOS8+
